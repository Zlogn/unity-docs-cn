Smooth Height

使用 Smooth Height 工具可以平滑高度贴图并柔化地形特征。在 Terrain Inspector 中，单击 Paint Terrain 图标，然后从地形工具列表中选择 Smooth Height。

Smooth Height 工具不会显著升高或降低地形高度，而是将附近区域平均化。此工具可以柔化景观并减少突然变化的外观。

例如，使用包含高频图案的画笔进行绘制后，平滑很有用。这些画笔图案往往会将尖锐的锯齿状边缘引入景观中，但可使用 Smooth Height 工具使这些粗糙外观柔化。
