粒子系统力场 (Particle System Force Field)

Particle System Force Field 组件将力应用于粒子系统的粒子。要将此组件附加到粒子系统，请在粒子系统中启用 External Forces 模块，并指定 Layer Mask 或特定的 Force Field 组件。

有关粒子系统及其用途的完整介绍，请参阅关于粒子系统的文档。

属性

使用 Particle System Force Field 组件将各种类型的力应用于粒子。

属性 功能
Shape
Shape 影响区域形状。
Start Range 形状的内部点，是影响区域的起点。
End Range 形状的外部点，是影响区域的终点。
Direction X, Y and Z 分别沿 x 轴、y 轴和 z 轴应用于粒子的线性力。该值越大，力就越大。可指定恒定力或随时间改变的力。请参阅随时间推移而变化的属性。
Gravity
Strength 粒子对形状内部聚焦点的吸引力。该值越大，强度就越大。可指定恒定强度或随时间改变的强度。请参阅随时间推移而变化的属性。
Gravity Focus 重力将粒子拉向的聚焦点。值 0 将粒子吸引到形状的中心，而值 1 将粒子吸引到形状的外边缘。
Rotation
Speed 粒子系统围绕涡旋推动粒子的速度（涡旋是力场的中心）。该值越大，速度越快。可指定恒定速度或随时间改变的速度。请参阅随时间推移而变化的属性。
Attraction 将粒子拖入涡旋运动的强度。值 1 表示最大吸引力，值 0 表示无吸引力。可指定恒定吸引力或随时间改变的吸引力。请参阅随时间推移而变化的属性。
Rotation Randomness 推动粒子围绕旋转的形状随机轴。值 1 表示应用最大随机性，值 0 表示不应用随机性。
Drag
Strength 阻力效果的强度，可以减缓粒子的速度。该值越大，强度就越大。可指定恒定强度或随时间改变的强度。请参阅随时间推移而变化的属性。
Multiply Drag by Size 根据粒子的大小调整阻力。
Multiply Drag by Velocity 根据粒子的速度调整阻力。
Vector Field
Volume Texture 矢量场的纹理。
Speed 应用于穿过矢量场的粒子的速度乘数。该值越大，速度越快。可指定恒定强度或随时间改变的强度。请参阅随时间推移而变化的属性。
Attraction Unity 将粒子拖入矢量场运动的强度。该值越大，吸引力就越大。可指定恒定吸引力或随时间改变的吸引力。请参阅随时间推移而变化的属性。
